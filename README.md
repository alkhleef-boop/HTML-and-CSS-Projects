# HTML and CSS 

## Overview

This is a comprehensive, colorful, and unique portfolio website designed for **Elham Al Khlef**. The project serves as a technical demonstration of modern HTML5 and CSS3 capabilities, featuring interactive elements, multimedia integration, and structured layouts.

---

## 🎨 Theme & Design
The website uses a **"Cyber-Vibrant"** aesthetic. It features a deep slate background with neon blue and pink accents to create a "cool" and high-energy feel. 

### Key Design Elements:
* **Color Palette:** Deep Navy (`#0f172a`), Sky Blue (`#38bdf8`), and Soft Pink (`#f472b6`).
* **Typography:** Clean, sans-serif fonts with bold weights for readability.
* **Interactivity:** Smooth transitions and hover states for a professional UI/UX feel.

---

## 🛠️ Technical Implementation

### 1. Navigation & Layout
* **Fixed Navbar:** Stays at the top of the page with a background color and links that change color on hover.
* **Two-Column Layout:** A balanced section using Flexbox to separate "Who I Am" and "My Skills," with custom text colors and fixed dimensions.
* **Bold Footer:** A centered, bolded footer for a professional close to the page.

### 2. Image Processing & Effects
* **Centered Image:** Aligned perfectly in the middle of the container.
* **CSS Filters:** Applied a `sepia` and `saturate` filter for a unique visual style.
* **Rounded Edges:** Utilized `border-radius` for a modern, soft-edged look.
* **Image Overlay Fade:** An interactive block where an image fades into a solid color with a text description when hovered over.
* **Text Over Image:** Positioned text directly on top of a background image for hero-style branding.

### 3. Multimedia Integration
* **Autoplay Video:** A high-quality background video that plays automatically on a loop.
* **Muted Audio:** The video is muted by default to ensure a non-disruptive user experience.
* **Video Text Overlay:** A large headline centered directly over the moving video background.

### 4. Data & Typography
* **Styled Table:** * **Borders:** Custom colored borders for a structured look.
    * **Cell Padding:** Generous spacing inside cells for better legibility.
    * **Top Row Styling:** A unique background color applied specifically to the header row.
* **Hover Paragraphs:** All `<p>` tags within specific sections feature a scaling and color-change effect when the mouse passes over them.
