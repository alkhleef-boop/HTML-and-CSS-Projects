# My HTML and CSS Projects
Welcome to my web development repository! This project showcases my journey through front-end development, specifically focusing on CSS styling, layouts, and the Bootstrap framework.

## 🚀 Projects Included

### 1. Bootstrap Cinema Project
A responsive theater website created using **Bootstrap 5**. 
- **Features:** Navigation bookmarks, card decks for movie listings, and info-colored pill badges for showtimes.
- **Movies Featured:** Kong: Skull Island, Ghost in the Shell, and Beauty and the Beast.

### 2. Personal Portfolio Website
A unique, "Cyber-Modern" themed portfolio built with **Pure HTML and CSS**.
- **Technical Implementation:**
  - **Fixed Navbar:** A sticky menu with hover effects.
  - **Multimedia:** Embedded autoplaying, muted background video with text overlays.
  - **CSS Mastery:** Includes image filters, rounded corners, and image overlay fade transitions.
  - **Layouts:** Features a two-column flexbox layout and a styled data table with custom borders and cell padding.
